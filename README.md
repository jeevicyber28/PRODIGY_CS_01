PRODIGY_CS_01

Simple Caesar Cipher Python program.

Files
- caesar_cipher.py: Encrypts and decrypts text using a Caesar Cipher.

How to push this project to GitHub
1. Create an empty repository on GitHub at: https://github.com/jeevicyber28/PRODIGY_CS_01
2. From the project root (d:\PRODIGY\jeevi prodigy\prodigy_cs-1) run:
   git init
   git add .
   git commit -m "Add Caesar Cipher program"
   git branch -M main
   git remote add origin https://github.com/jeevicyber28/PRODIGY_CS_01.git
   git push -u origin main

Usage
- Run: python caesar_cipher.py
